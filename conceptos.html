Introduccion a Framework CSS y JS(bootstrap)

HTML (Lenguaje de Marcado de Hipertexto): Lenguaje estándar para estructurar contenido en la web.

CSS (Hojas de Estilo en Cascada): Lenguaje de estilos para diseñar y mejorar la apariencia de las páginas web.

JavaScript: Lenguaje de programación para agregar interactividad y dinamismo a las páginas web.

Angular: Framework de JavaScript para construir aplicaciones web dinámicas y de una sola página.

React: Librería de JavaScript creada por Meta para construir interfaces de usuario interactivas.

Vue.js: Framework progresivo de JavaScript para construir interfaces de usuario y aplicaciones de una sola página.

Bootstrap: Framework de CSS que facilita la creación de sitios web responsivos y estilizados.

Tailwind CSS: Framework de CSS de utilidad que permite la creación de interfaces personalizadas y rápidas.

Foundation: Framework de diseño responsivo y accesible para la creación de sitios web modernos.

Diseño Responsivo: Diseño web que se adapta a diferentes tamaños de pantalla, garantizando una buena experiencia en cualquier dispositivo.

UI (Interfaz de Usuario): Diseño visual de una aplicación que incluye botones, menús y demás elementos con los que los usuarios interactúan.

UX (Experiencia de Usuario): Disciplina centrada en cómo se siente el usuario al interactuar con un producto o servicio.

Desarrollo Frontend: Desarrollo de la parte visible de las aplicaciones web, que interactúa directamente con el usuario.

Desarrollo Backend: Desarrollo de la parte oculta de una aplicación web que maneja la lógica, bases de datos y servidores.

Python: Lenguaje de programación de alto nivel utilizado en desarrollo web, análisis de datos, automatización y más.

PHP: Lenguaje de programación utilizado en desarrollo web, especialmente en el lado del servidor.

Node.js: Entorno de ejecución para JavaScript en el lado del servidor que permite crear aplicaciones web escalables.

.NET: Framework de desarrollo para aplicaciones en Windows, aplicaciones web, móviles, entre otros.

Base de datos SQL: Bases de datos relacionales que utilizan Structured Query Language (SQL) para manejar los datos.

Base de datos NoSQL: Bases de datos no relacionales diseñadas para almacenar grandes cantidades de datos no estructurados.

APIs: Conjunto de reglas que permiten que diferentes programas se comuniquen entre sí y compartan datos o funcionalidades.

Servidores web: Software que maneja las solicitudes HTTP y sirve contenido web a los navegadores de los usuarios.

Hosting: Servicio que permite almacenar y servir sitios web en internet, utilizando servidores remotos.

Dominio: Nombre único que identifica a un sitio web en Internet, como "google.com".

CMS: Plataforma que permite crear, gestionar y modificar contenido en un sitio web sin conocimientos de programación.

SEO: Conjunto de estrategias para mejorar la visibilidad de un sitio web en los motores de búsqueda.

CDN: Red de servidores distribuidos que entregan contenido de manera rápida y eficiente a los usuarios, mejorando el rendimiento.
